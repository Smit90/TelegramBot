# Telegram Bot
### Node js

#### Actions:
1. /movie {movie name}
2. /weather {city name}
3. /echo {message}

#### API:
1. http://www.omdbapi.com/ -- movie api
2. https://www.weatherapi.com/ -- weather api

#### npm library:
1. https://www.npmjs.com/package/node-telegram-bot-api
